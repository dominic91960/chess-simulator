# Chess Simulator

A visual chess position built to arrrange pieces on board, choose board orientation and export the board layout as a high-resolution JPEG.

## Features

- Drag pieces to any position on the board
- Add multiple pieces of the same kind from the piece menu
- Remove pieces by dragging them off the board/right-clicking the piece
- Clear the entire board
- Flip the board orientation
- Export the board layout as a 4× JPEG image
